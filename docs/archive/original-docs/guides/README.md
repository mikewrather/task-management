# User Guides

This directory contains user-facing documentation, setup guides, and workflow instructions.

## Contents

- **[user_guide.md](user_guide.md)** - Main user documentation and getting started guide

### Subdirectories

- **[setup/](setup/)** - Setup and configuration guides
  - Implementation guides
  - Device configuration (Pixel Watch, voice recorders)
  - Initial system setup

- **[workflows/](workflows/)** - Usage workflows and processes
  - Complete voice processing workflows
  - Automation configuration
  - File management and cleanup

- **[integrations/](integrations/)** - Integration-specific guides
  - Google Drive setup and configuration
  - Notion integration guides

## Purpose

Guides provide step-by-step instructions for using the voice task management system. These documents are designed for end users and system administrators.

Use these documents when:
- Setting up the system for the first time
- Configuring integrations
- Learning daily workflows
- Troubleshooting common issues
