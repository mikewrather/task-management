## Quick Start

For basic task management:
```
"Add task to review code"
"What are my next actions?"
"Create project for website redesign"
```

For note-taking:
```
"Take a note about the API changes"
"Meeting with Sarah about Q3 planning"
"I learned about React Server Components"
```

## Key Concepts

- **Projects**: Time-bound initiatives with specific outcomes
- **Areas**: Ongoing responsibilities requiring maintenance
- **Resources**: Knowledge and reference materials
- **Archive**: Completed projects and outdated information
